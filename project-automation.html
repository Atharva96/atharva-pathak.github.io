Why I Built This

Leadership needed daily sales updates and dashboard refresh status every morning. Manual reporting was time-consuming, error-prone, and lacked real-time visibility into pipeline health. This solution eliminates manual effort and ensures timely, accurate insights every day.

What I Built

  • End-to-end ETL orchestration using Azure Data Factory
  • Automated Power BI dataset refresh using REST API
  • WhatsApp Business API integration for daily delivery
  • Refresh monitoring with status tracking and alerts
  • Automated failure notifications for quick resolution
