Why I Built This

Business users relied on BI team for every ad-hoc sales query. Manual query handling caused delays in decision-making. Needed a self-service, governed solution for quick, accurate sales insights.

What I Built

  • AI Copilot using Copilot Studio with a child agent for NL to SQL conversion
  • Connected to enterprise data in Azure Synapse SQL pool
  • Secure execution with role-based data access and validation
  • Clean, conversational UI for business users
